# Delight Bakery Website

A modern, premium bakery website for Delight Bakery located in Jaipur, India. Built with Next.js, Tailwind CSS, and Framer Motion.

## 🎯 Features

- **Modern & Responsive Design**: Beautiful UI that works perfectly on all devices
- **Smooth Animations**: Powered by Framer Motion for delightful user experience
- **SEO Optimized**: Meta tags and semantic HTML for better search engine visibility
- **Fast Loading**: Optimized images and code splitting for lightning-fast performance
- **WhatsApp Integration**: Direct ordering through WhatsApp button
- **Google Maps**: Embedded location map for easy navigation
- **Premium Aesthetics**: Warm bakery colors (cream, pastel pink, brown, beige)

## 📄 Pages

1. **Homepage** - Hero section, featured products, bakery story, reviews, and CTA
2. **Menu** - Complete product catalog with filtering by categories
3. **Gallery** - Beautiful masonry grid layout showcasing bakery products
4. **About** - Bakery story, values, and philosophy
5. **Reviews** - Customer testimonials and ratings (4.4 stars, 410+ reviews)
6. **Contact** - Contact form, opening hours, and Google Maps integration

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Layout**: React Masonry CSS

## 📦 Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🏗️ Build for Production

```bash
npm run build
npm start
```
